TROUBLESHOOTING:

Q: "mingw32-make: command not found"
A: Install MinGW and add bin directory to PATH
   Or run: cd C:\MinGW\bin && mingw32-make all

Q: PowerShell script won't run
A: Run this first:
   Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
   Then: .\run_cosmic_animation.ps1

Q: Chapter windows not closing automatically
A: The script uses taskkill. If it doesn't work:
   - Close windows manually
   - Or use batch file instead

Q: Animation runs too fast/slow
A: Edit the $duration variable in PS script (default 18 seconds)
   Then rebuild: mingw32-make clean && mingw32-make all

Q: Some chapters missing/not found
A: Verify all .cpp files exist
   Run: mingw32-make clean && mingw32-make all
   Check: dir *.exe (should show 16 executables)
