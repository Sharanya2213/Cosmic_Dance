COSMIC DANCE - 5 MINUTE ANIMATION
=====================================

QUICK START:

1. Open PowerShell in the project directory:
   cd "c:\Users\Sharanya\OneDrive\Desktop\Cosmic Dance"

2. Run the animation:
   powershell -ExecutionPolicy Bypass .\run_cosmic_animation.ps1

3. The script will:
   ✓ Compile all 17 chapters
   ✓ Launch each chapter for ~18 seconds
   ✓ Auto-transition between chapters
   ✓ Display completion screen

=====================================

ALTERNATIVE METHODS:

Batch File (Simpler):
  run_cosmic_animation.bat

Manual Via Makefile:
  mingw32-make clean
  mingw32-make all
  mingw32-make run1  (then manually switch)

TROUBLESHOOTING:

Q: "mingw32-make not found"
A: Install MinGW and add it to PATH

Q: Script won't run in PowerShell
A: Try: Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process

Q: Chapters too fast/slow
A: Edit duration values in the PS1 script (default 18 seconds)

Q: Window position issues
A: This is normal - Windows stacks them. Use Alt+Tab to navigate.
